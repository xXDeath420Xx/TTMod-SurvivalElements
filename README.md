# Survival Elements

Adds machine health and repair mechanics to Techtonica.

## Features

- **Repair Tool** - Equipment for repairing damaged machines
- Machine health and repair mechanics integration
- Tech tree unlock for Repair Tool

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+
- TechtonicaFramework 1.0.0+

## Installation

Install via r2modman or manually place the DLL in your BepInEx/plugins folder.
