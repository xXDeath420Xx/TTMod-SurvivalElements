# Changelog

All notable changes to SurvivalElements will be documented in this file.

## [2.6.3] - 2026-01-07

### Changed
- Completely redesigned health/hunger UI for cleaner look
- Bars are now shorter (140px vs 280px) and thinner (18px vs 28px)
- Tight vertical stacking with minimal gap
- Positioned adjacent to the toolbar on the right side
- Compact text display (just the value, no "/ max")
- Warning messages now appear above the bars
- Smaller, more subtle font sizes throughout

## [2.6.2] - 2026-01-07

### Fixed
- Moved health/hunger UI higher to avoid overlapping with Blueprint UI and Steam achievements

## [2.6.1] - 2026-01-07

### Changed
- All items now appear in unified "Modded" build menu tab
- Updated dependency on TechtonicaFramework 1.2.0

### Fixed
- Build menu organization for cleaner UI

## [1.0.0] - 2025-01-05

### Added
- Initial release
- **Repair Tool** - Equipment for repairing damaged machines
- Machine health and repair mechanics integration
- Tech tree unlock for Repair Tool
